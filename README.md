# Fujifilm Model - Exiftool

Python script that interacts with the command line interface (CLI) to change the camera model on Fujifilm cameras ( especially older models ) to an X-T4
so the following simulations can be used:
* ACROS
* ETERNA
* SEPIA
* BLEACH BYPASS
* CLASSIC NEGATIVE

### Requirements
-- Python 3.6 or higher

-- Exiftool

-- Pandas

-- Tabulate

-- Typer

### Installation
```bash
pip3 install pandas
pip3 install tabulate
pip3 install typer
```
```bash
git clone https://github.com/exiftool/exiftool.git
```

### Virtual Environment
```bash
pip3 install pipenv
pipenv shell
pipenv install -r ./requirements.txt
```

## Usage
```bash
python3 exifcli.py
```
